IE warning - Doesn't appear unless you are using below IE 9

If you are using IE (in this case less than 9) you get an error bar that appears and tells you your browser is up to date, and provides links to upgrade links for IE (boo), Firefox, Safari and Chrome.

If you want to test this in your normal browser, comment out the if statement that uses bowser to check the browser version.
